1) Quais as possibilidades de cores e modos de colocá-las?
As possibilidades são de 256**3, ou seja, 16.777.216, essa conta é feita com base no padrão RGB (Red, Green e Blue), para cada um possuimos 256 possibilidades (De 0 a 255).
Colocamos as cores usando os modos de RGB(Usando o # + o código da cor, Ex: #FF00FF. Nessa caso, se o numero ou a letra se repetir, podemos fazer uma otimização... Ex: #FF00FF, irá virar #F0F). Além disso, a cor pode ser colocada com o uso do seu nome. Ex: "maroon", "blue", "green", etc...

2) Como pode ser feito para colocar cor de fundo em um site?
O cor de fundo de um site pode ser trocado utilizando "background-color:'Nome da cor, ou seu código' " dentro de um arquivo CSS ou fazendo a referencia "style" dentro do arquivo HTML.
